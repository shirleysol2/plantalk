## 2026-05-15 자동 반영과 계획노트 편집

- 채팅 문장에 `완료`, `확정`, `변경`, `바꾸자` 같은 표현이 있으면 할 일/결정/일정 상태를 더 적극적으로 자동 반영하도록 추론 규칙을 보강했다.
- 채팅 말풍선에서 우클릭, 길게 누르기, 더보기 버튼으로 `일정/할 일/결정/예산으로 반영` 메뉴를 열 수 있게 했다.
- 수동 반영 시 어떤 사용자 코드가 메시지를 계획 항목으로 반영했는지 `plantalk.infoLogs`에 `message_applied` 로그를 남긴다.
- 계획노트 안에서 일정, 할 일, 결정사항, 예산 항목을 직접 수정하거나 삭제할 수 있게 했다.
- 결정사항은 계획노트에서 바로 `확정` 상태로 바꿀 수 있게 했다.
